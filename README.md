# Piece

Each bit represents the shape of a piece.

- 3th bit: Black(1) or White(0)
- 2th bit: Hole(1) or No-hole(0)
- 1th bit: Round shape(1) or Square shape(0)
- 0th bit: High(1) or Low(0)

```
example: 
piece 10 = 1010 = (Black + No-hole + Round + Low) piece
```
